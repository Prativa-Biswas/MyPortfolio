ETL Process in Traditional Way (Flexcube Data Integration & Transformation)

Project Overview:

Worked with pre-defined PL/SQL packages to extract data from the Flexcube environment into an Oracle database using DB links. Processed data through a 3-layer package architecture to populate Fact and Dimension tables.

✅Technologies Used:

Oracle PL/SQL, SQL, DB Links, Stored Procedures, Triggers, Packages

Key Responsibilities:

  • Extracted data from Flexcube via DB links and loaded it into staging tables.
  • Utilized pre-existing PL/SQL packages to process and transform data.
  • Monitored and optimized query performance for efficient data processing
  • Ensured data accuracy and consistency in the final Fact and Dimension tables

Bank Application – Data Import & Processing

Project Overview:

Imported client-provided Excel data into Oracle tables and processed it using PL/SQL packages to meet business requirements.

✅Technologies Used:

Oracle PL/SQL, SQL Loader, Excel Data Import, Stored Procedures, Triggers, Packages

Key Responsibilities:

  • Data Import: Converted and imported structured Excel data into Oracle tables.
  • Custom Business Logic: Developed PL/SQL packages to process the data based on client-specific rules
  • Data Validation & Transformation: Ensured data accuracy by implementing validation checks and transformations before final processing.
  • Automated Processing: Streamlined the execution of PL/SQL packages for efficient data handling.
