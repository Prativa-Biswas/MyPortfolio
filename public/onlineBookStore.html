Online Book Store

Project Overview:

Developed Online Book Store which is a web-based application designed to provide a seamless platform for users to browse, search, purchase, and manage books online. The system efficiently handles book inventory, order processing, and user authentication to ensure a smooth shopping experience.

✅Technologies Used:

✔ Backend: Core Java, JDBC, Servlets, JSP
✔ Database: Oracle (SQL, PL/SQL)
✔ Frontend: HTML, CSS
✔ Server & Deployment: Tomcat Server

Key Features & Responsibilities:

  • Inventory Management: Designed a database-driven system to store and manage books, including categories, authors, prices, and stock.
  • User Authentication: Implemented secure login/logout functionality for user account management.
  • Order Processing: Developed an efficient cart, order placement, and checkout system.
  • Database Optimization: Created a normalized schema and wrote optimized SQL queries, stored procedures, and triggers for real-time data retrieval.
  • Seamless Deployment: Deployed the application on a Tomcat server ensuring smooth performance.
